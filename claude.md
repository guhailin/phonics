## About the Summarize Skill

The `summarize` skill automatically analyzes:
- `README.md` for project overview
- Git status for file changes
- Key source files for technical details
- Recent commits for change history

It generates comprehensive markdown documentation keeping both `agent.md` and `claude.md` in sync.
