# 🎨 Oxford Phonics World - 儿童自然拼读学习网页

一个专为儿童设计的卡通风格自然拼读（Phonics）学习网页应用，基于Oxford Phonics World教材（Level 2-5）。

## 📚 项目简介

本项目是一个纯前端的交互式学习应用，帮助儿童通过视觉化、游戏化的方式学习英语自然拼读规则。覆盖从基础短元音到复杂字母组合的完整学习路径，包含240个精选单词和2400+探索模式单词。

## ✨ 核心功能

### 1. **四级渐进式学习**
- 🟢 **Level 2 - Short Vowels（短元音）**：学习CVC结构，包含6个单元60个单词
- 🔵 **Level 3 - Long Vowels（长元音）**：掌握Magic E和长元音组合，6个单元60个单词
- 🟡 **Level 4 - Consonant Blends（辅音混合音）**：学习辅音连读技巧，6个单元60个单词
- 🔴 **Level 5 - Letter Combinations（复杂组合）**：掌握R控制元音和特殊组合，6个单元60个单词

### 2. **三层导航结构**
- **首页**：展示4个级别的彩色卡片，带悬浮动画效果
- **Level详情页**：显示该级别下所有单元，提供复习和探索模式入口
- **Unit详情页**：展示单词网格，每个单词配有：
  - 🎨 生动的emoji图标（240个独特图标）
  - 🔤 自动高亮的关键字母（如：c**at**、tr**ai**n）
  - 📱 响应式卡片布局

### 3. **探索模式（Level层级）**
- 🔍 海量单词练习：自动收集整个Level所有单词（约600个/级别）
- 📝 纯文字展示，专注拼读规则，无图片干扰
- ✨ 自动高亮关键字母组合，与学习单词一致
- 🎲 支持随机打乱和快速切换
- ⌨️ 键盘快捷键支持：
  - `→` 或 `空格`：下一个单词
  - `←`：上一个单词
  - `R`：重新随机打乱
- 📊 实时进度显示（如：350/600）
- 💪 以简单常见单词为主，适合大量练习巩固

### 4. **智能复习模式**
- 📖 自动收集当前Level的所有单词
- 🔀 随机打乱顺序，避免记忆模式
- ⏭️ 支持上一个/下一个单词切换
- ⌨️ 键盘快捷键支持：
  - `→` 或 `空格`：下一个单词
  - `←`：上一个单词
  - `R`：重新随机打乱
- 📊 实时进度显示（如：15/60）

### 5. **视觉设计特色**
- 🎨 渐变紫色背景，营造梦幻氛围
- 🟢🔵🟡🔴 每个Level有独特的主题色
- 🎭 圆润卡通字体（Comic Sans MS）
- ✨ 丰富的动画效果：
  - 页面切换淡入动画
  - 卡片悬浮和弹跳效果
  - emoji浮动动画
  - 复习卡片弹出动画
- 🌈 关键字母渐变高亮（黄→红）

## 🎯 技术特点

### 前端技术栈
- **纯原生JavaScript**：无任何框架依赖，轻量高效
- **CSS3动画**：流畅的过渡和变换效果
- **响应式设计**：完美适配桌面、平板和手机
- **语义化HTML5**：结构清晰，易于维护

### 核心算法
1. **智能高亮算法**：支持普通高亮和Magic E（如i_e, o_e）模式识别
2. **Fisher-Yates洗牌算法**：确保复习单词的真随机性
3. **状态管理**：使用全局变量管理当前Level/Unit/Review状态

### 交互设计
- **三层导航**：首页→Level→Unit，层次清晰
- **流畅动画**：300ms过渡时间，符合人眼感知
- **即时反馈**：悬停、点击都有视觉响应
- **无障碍设计**：支持键盘操作，字体清晰易读

## 🔧 自定义扩展

### 添加新单词
编辑 `data.js`，在对应Level的units数组中添加：
```javascript
{ word: 'example', highlight: 'am', emoji: '🎯' }
```

### 修改颜色主题
编辑 `style.css` 中的颜色变量，统一更换主题色。

### 添加音频
可扩展每个单词对象添加 `audio` 字段，在 `app.js` 中添加播放逻辑。
