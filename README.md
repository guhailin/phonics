# 🎨 Oxford Phonics World - 儿童自然拼读学习网页

一个专为儿童设计的卡通风格自然拼读（Phonics）学习网页应用，基于Oxford Phonics World教材（Level 1-5）。

## 📚 项目简介

本项目是一个纯前端的交互式学习应用，帮助儿童通过视觉化、游戏化的方式学习英语自然拼读规则。覆盖从字母表基础到复杂字母组合的完整学习路径，包含768个精选单词、640条例句和820个探索模式单词。

## ✨ 核心功能

### 1. **五级渐进式学习**
- 🟣 **Level 1 - Alphabet & Sounds（字母与发音）**：学习26个字母及其基本发音，8个单元，78个精选单词
- 🟢 **Level 2 - Short Vowels（短元音）**：学习CVC结构，8个单元142个单词
- 🔵 **Level 3 - Long Vowels（长元音）**：掌握Magic E和长元音组合，8个单元128个单词
- 🟡 **Level 4 - Consonant Blends（辅音混合音）**：学习辅音连读技巧，8个单元201个单词
- 🔴 **Level 5 - Letter Combinations（复杂组合）**：掌握R控制元音和特殊组合，8个单元219个单词

### 2. **三层导航结构**
- **首页**：展示5个级别的彩色卡片，带悬浮动画效果
- **Level详情页**：显示该级别下所有单元，提供复习和探索模式入口
- **Unit详情页**：展示单词网格（Level 1不含例句，Level 2-5包含例句练习）
  - 📚 **单词卡片Tab**：每个单词配有emoji图标和高亮字母
  - 💡 **例句练习Tab（Level 2-5）**：10个精心设计的例句，支持前后翻页
  - 🎨 生动的emoji图标（768个独特图标）
  - 🔤 自动高亮的关键字母（如：c**at**、tr**ai**n）
  - 📱 响应式卡片布局

### 3. **例句练习系统（Level 2-5）**
- 💡 **640条精选例句**：每个Unit包含20条例句（Level 1无例句）
- 🎯 **Pattern高亮**：例句中的关键拼读模式自动高亮并下划线
- 📖 **翻页浏览**：支持上一句/下一句切换，含进度显示（如：3/10）
- 👶 **儿童友好**：句子简短易懂，贴近儿童生活场景
- ✨ **视觉强化**：渐变色高亮关键字母，加深记忆

### 4. **探索模式（Level层级）**
- 🔍 海量单词练习：自动收集整个Level所有单词（约600个/级别）
- 📝 纯文字展示，专注拼读规则，无图片干扰
- ✨ 自动高亮关键字母组合，与学习单词一致
- 🎲 支持随机打乱和快速切换
- ⌨️ 键盘快捷键支持：
  - `→` 或 `空格`：下一个单词
  - `←`：上一个单词
  - `R`：重新随机打乱
- 📊 实时进度显示（如：350/600）
- 💪 以简单常见单词为主，适合大量练习巩固

### 5. **智能复习模式**
- 📖 自动收集当前Level的所有单词
- 🔀 随机打乱顺序，避免记忆模式
- ⏭️ 支持上一个/下一个单词切换
- ⌨️ 键盘快捷键支持：

## 🎯 技术特点

### 前端技术栈
- **纯原生JavaScript**：无任何框架依赖，轻量高效

### 交互设计

- **流畅动画**：300ms过渡时间，符合人眼感知
- **即时反馈**：悬停、点击都有视觉响应
- **无障碍设计**：支持键盘操作，字体清晰易读

## 🔧 自定义扩展

### 添加新单词
编辑 `data.js`，在对应Level的units数组中添加：
```javascript
{ word: 'example', highlight: 'am', emoji: '🎯' }
```

### 添加新例句
编辑 `data.js`，在对应Unit的examples数组中添加：
```javascript
examples: [
  "The <span class='highlight'>c<span class='pattern'>at</span></span> is on the mat.",
  "I have a pet <span class='highlight'>c<span class='pattern'>at</span></span>."
]
```
**注意**：使用 `<span class='highlight'>` 包裹完整单词，`<span class='pattern'>` 包裹关键Pattern

### 修改颜色主题
编辑 `style.css` 中的颜色变量，统一更换主题色。

### 添加音频
可扩展每个单词对象添加 `audio` 字段，在 `app.js` 中添加播放逻辑。
