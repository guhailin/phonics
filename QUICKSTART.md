## 学习流程

```
1. 首页 → 选择 Level (2-5)
2. Level页 → 选择 Unit
3. Unit页 → 切换Tab学习
   ├── 📚 单词卡片 (浏览所有单词)
   └── 💡 例句练习 (在句子中学习)
4. 返回Level页 → 
   ├── 🎯 开始复习 (测试掌握情况)
   └── 🔍 探索模式 (海量单词练习)
```

## 键盘快捷键 ⌨️

### 复习模式 & 探索模式
- `空格` 或 `→` : 下一个单词
- `←` : 上一个单词  
- `R` : 重新随机打乱

### 例句练习
- `→` : 下一句
- `←` : 上一句

## 学习建议 💡

1. **循序渐进**：从 Level 2 开始，不要跳级
2. **重复练习**：每个Unit的单词至少复习3遍
3. **结合例句**：在例句中理解单词用法
4. **探索模式**：完成一个Level后，用探索模式巩固
5. **关注高亮**：重点观察高亮的字母组合规律

## 常见问题

**Q: 可以离线使用吗？**  
A: 可以！下载整个文件夹后，无需网络即可使用。

**Q: 如何判断学习效果？**  
A: 在复习模式中，如果能快速认读并说出单词含义，说明掌握良好。

**Q: 支持哪些浏览器？**  
A: 支持所有现代浏览器（Chrome、Firefox、Safari、Edge）。
